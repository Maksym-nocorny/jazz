## Variant A — "Origin-first editorial" ✅ chosen (+ edit)
**Idea:** одразу показати, що це майстри й що походження видно — у дусі журналу.
**Section order (after edit):**
1. Header (лого ВОВК · Кава/Обсмажка/Гуртом/Журнал/Контакти · CTA «Обрати каву»)
2. Hero — асиметричний: H1 «Кава для тих, хто чує різницю» + підзаголовок ліворуч, велике фото
   зерна праворуч; CTA «Обрати каву» + ghost «Як ми обсмажуємо»; дрібний підпис про доставку
3. **Кава — 3 поточні лоти** (підняті вище за правкою клієнта; асиметричний список, не 3 картки)
4. Принципи — «Без компромісів…» (асиметрично, 3 пункти різного розміру: малі партії / прозоре
   походження / 48 годин)
5. Історія «Чому вовк» — темний espresso story-band, фото + підпис Андрія
6. Підписка / Гурт — спарена CTA-секція (підписка + «для кав'ярень»)
7. Розсилка (footer-форма) + Footer
**Pros:** найшвидше доносить позиціонування (дані=довіра); 3 лоти близько до верху = ранній шлях
до покупки; асиметрія прямо втілює напрямок і уникає AI-сітки.
**Cons:** герой має зробити багато роботи; ризик, що принципи «провисають» нижче лотів (рішення:
тримати їх короткими).
**Self-score vs goal (продаж + заявки): 9/10.** Найкращий баланс «віра в майстра» + ранній CTA.

## Variant B — "Shop-first"
**Idea:** максимально швидко до покупки; герой = одразу вітрина.
**Section order:**
1. Header  2. Короткий герой-смужка (1 рядок + CTA)  3. **Сітка товарів одразу** (3 лоти + «Уся
кава»)  4. Принципи  5. Підписка/Гурт  6. Історія (стисло, внизу)  7. Розсилка + Footer
**Pros:** найкоротший шлях до кошика; добре для тих, хто вже знає бренд (трафік з Instagram).
**Cons:** жертвує «вірою в майстра» — без історії та даних зверху легко сприйняти як черговий
магазин (ризик злитися з архетипом B/C з research); герой без точки зору тяжіє до AI-патерну
«центр + кнопка».
**Self-score vs goal: 7/10.** Конверсійно, але слабше на довірі/диференціації — головній загрозі.

## Variant C — "Story-first"
**Idea:** спершу емоція й бренд вовка, купівля — як наслідок довіри.
**Section order:**
1. Header  2. Великий редакторський герой про вовка/майстерню  3. Історія «Чому вовк» (розгорнуто)
4. Принципи  5. **Кава — 3 лоти**  6. Підписка/Гурт  7. Розсилка + Footer
**Pros:** найсильніша атмосфера й бренд; добре для «холодного» відвідувача.
**Cons:** покупка задовго відсунута вниз — прямо б'є по головній метриці (50 замовлень/міс);
для мобільного трафіку з діректа забагато скролу до товару.
**Self-score vs goal: 6/10.** Гарно, але повільно до дії.

## Recommendation
**A — Origin-first editorial.** Він єдиний тримає обидві цілі разом: рання довіра через дані
походження *і* ранній шлях до покупки, у точному дусі напрямку (асиметрія, журнальність). B надто
«магазинний» і ризикує злитися; C відсуває покупку.

**Client edit applied:** підняти секцію «Кава — 3 лоти» вище (над «Принципами»), щоб товар
з'являвся одразу після героя. Структуру вище наведено **вже з цією правкою** (round 2, approved).
Канонічний порядок секцій для білда — як у Variant A вище.
